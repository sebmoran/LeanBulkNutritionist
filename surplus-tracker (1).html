First-time setup — create the Supabase table

Run this SQL once in your Supabase dashboard → SQL Editor before using the app:

create table if not exists weekly_logs ( id bigint primary key generated always as identity, created_at timestamptz default now(), log_date date not null unique, weight_trend numeric(5,3) not null, -- kg/week (HappyScale trend) waist_cm numeric(5,1) not null -- cm ); alter table weekly_logs enable row level security; create policy "anon all" on weekly_logs for all using (true) with check (true);
